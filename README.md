# Navneet Sukhraj - Video Editing Portfolio

A React frontend portfolio for showcasing video editing and motion graphics work.

## Tech Stack

- React 19
- Tailwind CSS
- shadcn/ui with Radix UI
- CRACO
- Vercel

## Quick Start

```bash
yarn install
yarn start
```

## Build

```bash
yarn build
```

## Main Files

- `src/App.js` - main application
- `src/components/` - page sections and UI components
- `src/data/mock.js` - portfolio content
- `src/index.css` - global styles and theme variables
